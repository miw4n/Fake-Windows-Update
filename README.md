## Description

**Français :**  
Fake Windows Update est une simulation réaliste d'un écran de mise à jour Windows. Vous pouvez choisir la durée, activer le plein écran, afficher les étapes de mise à jour et un BIOS POST simulé. L'animation du spinner et le pourcentage progressent de manière réaliste, avec un faux redémarrage à la fin.

**English :**  
Fake Windows Update is a realistic simulation of a Windows update screen. You can set the duration, enable fullscreen, show update steps, and display a fake BIOS POST. The spinner animation and percentage progress realistically, with a fake restart at the end.

## Features / Fonctionnalités

- Réglage de la durée de la mise à jour / Set the update duration  
- Mode plein écran automatique / Auto fullscreen mode  
- Spinner animé avec pourcentage / Animated spinner with percentage  
- Faux BIOS POST après mise à jour / Fake BIOS POST after update  
- Possibilité d’annuler avec `Esc` / Cancel with `Esc` key  
- Texte et étapes réalistes inspirés de Windows / Realistic Windows-inspired text and steps  

## How to use / Comment utiliser

1. Ouvrir `index.html` dans votre navigateur  
   / Open `index.html` in your browser  
2. Configurer les options : durée, plein écran, afficher les étapes  
   / Configure options: duration, fullscreen, show steps  
3. Cliquer sur **Start Fake Update**  
   / Click **Start Fake Update**  
4. Attendre la progression et le faux redémarrage  
   / Wait for the progress and fake restart  

Appuyez sur **Esc** pour revenir au menu à tout moment.  
/ Press **Esc** to return to the menu at any time.

## Requirements / Prérequis

- Navigateur moderne compatible HTML5, CSS3 et JavaScript  
- Modern browser supporting HTML5, CSS3, and JavaScript
